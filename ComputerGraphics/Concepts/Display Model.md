 **Definition**:
A display model is the **representation of graphical objects in a form that is ready to be rendered on the screen**.

It describes how objects appear visually — including their geometry, color, transformations, and other rendering properties.


### **Core Idea**
In Graphics Systems, we often separate:
- [[Application Model]] -> Stores logical / real world data
- **Display Model** -> stores visual / drawable representation

The display model is what is used when the scene is actually drawn


---
*Think [[Application Model]] is the **Data Structure** describing what the world is*
***Display Model** is the graphical version of that data ready for rendering*

---

### **What the Display Model Typically Contains**
- Geometric data (vertices, shapes)
- Transformations
- Visual properties
- Rendering state information
