
**Definition**
A discrete image is an image represented as a **finite grid of pixels**, where each pixel stores a quantized color value at integer coordinates.

It is a sampled approximation of a continuous visual scene.


### **Core Idea**

- Defined only at integer pixel locations (x , y)
- Stored as a 2D array in memory
- Each pixel contains a finite (quantized) color value
- Forms the basics of [[Raster Graphics]] and the [[Frame Buffer]]


### **What Makes it Discrete**

1. Spatial Discretization
	- The image only exists at grid points
	- No information about the space between pixels or points

2. Intensity (Color) Discretization
	- Colors are limited to fixed values
	- No infinite precision colors

**General Idea:** Finite, not Infinite
