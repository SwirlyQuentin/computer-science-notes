Scalable Vector Graphics

**Definition**:
SVG is an XML-based file format for describing 2D vector graphics.

**Key Points:**
- Used primarily in web development
- Resolution-independent (scales without losing quality)
- Defined using shapes, paths, text, and transformations
- Can be styled with CSS and manipulated with JavaScript
- Also follows a [[Retained Mode | retained mode model]]

**In Graphics Context**:
Instead of storing pixels (like raster images), SVG stores **mathematical descriptions of shapes**.
