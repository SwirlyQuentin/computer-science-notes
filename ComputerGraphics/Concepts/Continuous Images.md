**Definition:**
A continuous image is an image represented as a **mathematical function defined over continuous (real-valued) coordinates**, where intensity or color exists at every possible point in space.

It represents the ideal, infinite-detail version of a visual scene.


### **Core Idea**

- Defined for all real coordinates (x, y)
- Infinite spatial resolution
- No pixels
- Used as the theoretical model before [[Sampling | sampling]]

**INFINITE** not finite

### **Key Properties**

- Infinite detail 
- Infinite precision
- No resolution limit
- Not storable on a computer
