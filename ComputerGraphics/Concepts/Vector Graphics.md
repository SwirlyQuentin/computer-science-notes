Definition:
Vector graphics represent images using **mathematical equations and geometric primitives** (points, lines, curves, shapes) instead of pixels.

### **Core Idea**

Vector graphics store:
- Coordinates
- Lines
- Curves
- Shapes
- Transformations

The image is generated **Mathematically** when rendered

**Characteristics:**
- Resolution **Independent
- scales infinitely without losing quality
- Ideal for logos, UI, diagrams, and fonts

### **Common Formats**
- [[SVG]] (.svg)
- PDF (.pdf) (can contain vector)
- AI (.ai)
