Windows Presentation Foundation

**Definition**: WPF is a Microsoft framework for building desktop applications on Windows that use modern graphics, animations, and UI elements.

**Key Points**: 
 - Part of .NET ecosystem
 - uses XAML to define UI layouts
 - based on [[Retained Mode | retained mode graphics system]]
 - supports vector graphics, 2D, 3D, animation, and data binding
 - Renders using [[DirectX]]

**In Computer Graphics Context**: 
WPF stores and manages graphical objects internally (scene graph approach), rather than redrawing everything manually each frame.
