# **Random Noise**
Perlin noise uses random numbers to generate random looking textures
- Gives value at a certain pixel coordinates
- Perlin noise can be rescaled and added to itself to create fractal noise

Perlin Noise
- procedural texture primitive
- gradient noise used to increase the appearance of realism in computer graphics
- Has a pseudo-random appearance
	- Same seed equals same results
	- Nothing is truly random
- This allows perlin noise to be readily controllable

# **Using Perlin**
- As a source for the random noise, we need an array of random values called `noise[x][y]`
- 2D textures = 2D array
- The function generateNoise will fill the array with noise and the main function is programmed to show this noise on the screen
- The noise itself is generated with the rand() function 
	- This gives a value between 0-32678 which is then divided by 32768 to get a floating point number between 0 and 1

# **Turbulence**
- Turbulence is what creates natural looking features out of smoothed noise
- Trick is to add multiple noise textures of different zooming scales together

Example using mountains:
- Main mountains have very large features and jagged
- Then added to mountains are smaller features like variation in slope, hills, etc
- At an even smaller scale there are rocks on the mountain
- Even smaller is grains of sand

In 2D this is done by adding different sizes of the smoothed noise together
- Zooming factor can start at 16 and is divided by 2 each time until the zooming factor is 1
- In the mountain example, the small features werent only small in width but also height
- To achieve this in 2D textures, we can make the images with a smaller zoom darker 
	- Adding them will now have a less effect on the overall noise

By adding the 5 images together and dividing result through 5 to get the average, you get a turbulence texture
- This turbulence texture looks natural like clouds

# **Marble**
- Its possible to use random noise to create a textire that looks like marble
- By taking a sin pattern as a base or seed
- Then taking initial turbulence values and periods for the sin function you can get a marble like texture

# **Wood**
- Adding turbulence to the sin of the distance of x and y gives a wood like texture

# **Simplex Noise**
- We will be using a variation of Perlin Noise, Simplex noise
- Simplex Noise has the same purpose but uses a simpler space filling grid
- Also alleviates computational complexity and artifacting

# **How We Use it**
- We will use it to give randomization to the height of a chunk
- Provides random generation to terrain

# **How**
- Combine several octaves of noise to create fractal noise
- For each octave you divide the coordinates by frequency and multiplythe result by amplitude
	- This gives a terrain like appearance
- Persistence is used to affect the appearance of the terrain
	- High peristence gives rocky mountainous terrain
	- Low persistence gives varying flat terrain

To use in program:
- Create simplex noise object inside out rebuildMesh
- Simplex noise requires 3 arguments
	- "Largest Feature"
	- Persistence
	- Seed
The simplex noise class creates octaves that gives features of size between 1 and largestFeature
