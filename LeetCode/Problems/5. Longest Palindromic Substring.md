
## **My Code**
```cpp
class Solution {
public:
    string longestPalindrome(string s) {
        string longestPalindromic = "";
        string workingSubstring = "";

        for (int i = 0; i < s.length(); i++){
            for (int j = i; j < s.length(); j++){
                workingSubstring += s.at(j);

                if (isPalindrome(workingSubstring)){
                    if (workingSubstring.length() > longestPalindromic.length()){
                        longestPalindromic = workingSubstring;
                    }
                }
            }

            workingSubstring = "";
        }

        return longestPalindromic;
    }

    bool isPalindrome(string s){
        int n = s.length() / 2;

        string ending = s.substr(s.length() - n, n);
        reverse(ending.begin(), ending.end());

        if (s.substr(0, n) == ending){
            return true;
        }
        else{
            return false;
        }
    }
};
```

#### **Issues**
- Brute force method
- Results in extremely high time complexity
- calling isPalindrome on every substring

#### **Time Complexity**
- Time complexity is $O(n^3)$
- Time complexity of palindrome checking is $O(n)$
- **VERY SLOW**

## **Solution**

1. Loops through string and checks left + right
2. If left and right are equal, whole thing is palindrome
3. increase left and right by 1 so see if the next increment is also a palindrome

#### **Time Complexity**
- The time complexity is $O(n^2)$
- The time complexity of palindrome checking is $O(1)$

#### **Odd Palindromes**
If the palindrome is odd then an index is determined as the center, and l + r expands outwards to check palindrome

##### Example: Expanding Around Center

**Initial index**

- i = 2

```
"abcba"
  l i r
```

---

##### Step 1

l = 1  
r = 3  

Check:
- left = "b"
- right = "b"

"bcb" is a palindrome 😆

Let's expand the range!

```
"abcba"
 l i r
```

---

##### Step 2

l = 0  
r = 4  

Check:
- left = "a"
- right = "a"

"abcba" is a palindrome 😆

max_length = 5 ("abcba")

```
"abcba"
 l  i  r
```

---

##### Step 3 (Out of Bounds)

l = -1  
r = 5  

Left and right are now **out of bounds**, so the iteration stops.


#### **Even Palindromes**

When the palindrome we are looking for is even, we pick either l or r to be the center and expand out

##### Example: Even-Length Palindrome Expansion

Current index = 1

```
  l r
"abbc"
  i
```

We found palindrome **"bb"**.

```
 l   r
"abbc"
  i
```

Finish iteration.


### Algorithm Detailed Explanation

1. **Check for empty string**

   - If the input string `s` is empty, return `""`.
   - An empty string cannot contain a palindromic substring.

2. **Helper function**

   - Define a helper function `expand_around_center(s, left, right)`.
   - This function expands outward while:
     - `left >= 0`
     - `right < len(s)`
     - `s[left] == s[right]`
   - It returns the **length of the palindrome** found.

3. **Initialize tracking variables**

   - Initialize:
     - `start = 0`
     - `end = 0`
   - These track the **indices of the longest palindrome found so far**.

4. **Iterate through the string**

   - Loop through each index `i` in the string `s`.

5. **Check odd-length palindrome**

   - Call:

   ```
   expand_around_center(s, i, i)
   ```

   - This treats `i` as the **center character**.

6. **Check even-length palindrome**

   - Call:

   ```
   expand_around_center(s, i, i + 1)
   ```

   - This treats the **gap between characters** as the center.

7. **Store palindrome lengths**

   - Store results in:

   ```
   odd
   even
   ```

8. **Determine the larger palindrome**

   - Compute:

   ```
   max_len = max(odd, even)
   ```

9. **Update longest palindrome**

   - If:

   ```
   max_len > (end - start)
   ```

   - Update indices:

   ```
   start = i - (max_len - 1) // 2
   end = i + max_len // 2
   ```

10. **Continue scanning**

- Repeat the process until all characters in `s` are processed.

11. **Return the result**

- Return the longest palindromic substring:

```
s[start:end + 1]
```

- The slice includes both the `start` and `end` characters.


### **Solution**
```cpp
class Solution {
public:
    string longestPalindrome(string s) {
        // Edge case: empty string
        if (s.empty()) {
            return "";
        }

        int start = 0;
        int end = 0;

        // Iterate through each character in the string
        for (int i = 0; i < s.length(); i++) {

            // Check odd-length palindrome (center = i)
            int odd = expandAroundCenter(s, i, i);

            // Check even-length palindrome (center between i and i+1)
            int even = expandAroundCenter(s, i, i + 1);

            int max_len = max(odd, even);

            // Update longest palindrome boundaries
            if (max_len > end - start) {
                start = i - (max_len - 1) / 2;
                end = i + max_len / 2;
            }
        }

        // Return the longest palindromic substring
        return s.substr(start, end - start + 1);
    }

private:
    // Expands outward from the center while characters match
    int expandAroundCenter(string s, int left, int right) {
        while (left >= 0 && right < s.length() && s[left] == s[right]) {
            left--;
            right++;
        }

        // Length of the palindrome
        return right - left - 1;
    }
};
```
