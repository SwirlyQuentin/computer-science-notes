
### 1. Noise Library

Get FastNoiseLite from github + add to /include
```cpp
#include "FastNoiseLite.h"
```


### 2. Create Noise Object

Create the noise object and set the parameters frequency + seed + type
We are using perlin noise

```cpp
FastNoiseLite noise;
noise.SetNoiseType(FastNoiseLite::NoiseType_Perlin);
noise.SetFrequency(0.1f);
noise.SetSeed(1337);
```


### 3. Update getHeight Function

Update getHeight function to return a noise value based on the xz value 

```cpp
float getHeight(FastNoiseLite& noise, int x, int z) {
    float n = noise.GetNoise((float)x, (float)z); // returns [-1, 1]
    return n * 6.0f; // scale to [-6, 6] world units
}
```

Update inside the world gen loop:
```cpp
float y = getHeight(noise, x, z);
```


### 4. Make the Grid Bigger

Making the grid bigger to allow the noise function to control more + make it look more like terrain

```cpp
const int GRID_WIDTH = 32;

```

adjust camera
```cpp
glm::vec3 cameraPos = glm::vec3(16.0f, 14.0f, 38.0f);
```


### Pitfalls
- **Frequency controls zoom** - 0.1f gives broad hills, higher values give jagged noise
- **The Noise returns [-1, 1]** scale the noise by an arbitrary tuning number
- Don't construct the noise object per call as it is wasteful and will reseed every frame
