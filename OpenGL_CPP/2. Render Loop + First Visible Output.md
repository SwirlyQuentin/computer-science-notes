**Goal:** Set up render loop, window resizing, and get colors displayed to the screen

### 1. Set viewport
- Setting the viewport for things to be rendered into

```cpp
glViewport(0, 0, 800, 600);
```


### 2. Handle Window Resize
- Sets a callback function thats called when the window is resized

Function:
```cpp
void framebuffer_size_callback(GLFWwindow* window, int width, int height) {
    glViewport(0, 0, width, height);
}
```

Setting it:
```cpp
glfwSetFramebufferSizeCallback(window, framebuffer_size_callback);
```


### 3. Add Clear Color
- Inside render loop, coloring background of the window a certain color

```cpp
glClearColor(0.2f, 0.3f, 0.3f, 1.0f);
glClear(GL_COLOR_BUFFER_BIT);
```


### 4. Basic Input
- Uses the escape key to exit the program when pressed
- Lives in the render function

```cpp
if (glfwGetKey(window, GLFW_KEY_ESCAPE) == GLFW_PRESS)
    glfwSetWindowShouldClose(window, true);
```


### 5. Enable VSync
- Enables vsync to sync the framerate with the refreshrate

```cpp
glfwSwapInterval(1);
```
