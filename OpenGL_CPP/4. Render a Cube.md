
### 1. Replace Triangle with Cube Vertex Data

Replacing triangle vertices with cube vertices locations
```cpp
float vertices[] = {
    // positions        
    -0.5f, -0.5f, -0.5f,  // 0
     0.5f, -0.5f, -0.5f,  // 1
     0.5f,  0.5f, -0.5f,  // 2
    -0.5f,  0.5f, -0.5f,  // 3
    -0.5f, -0.5f,  0.5f,  // 4
     0.5f, -0.5f,  0.5f,  // 5
     0.5f,  0.5f,  0.5f,  // 6
    -0.5f,  0.5f,  0.5f   // 7
};
```


### 2. Add Indices

Group of vertices that make the square 
Are in groups of triangles

```cpp
unsigned int indices[] = {
    // back face
    0, 1, 2,
    2, 3, 0,

    // front face
    4, 5, 6,
    6, 7, 4,

    // left
    4, 7, 3,
    3, 0, 4,

    // right
    1, 5, 6,
    6, 2, 1,

    // bottom
    4, 5, 1,
    1, 0, 4,

    // top
    3, 2, 6,
    6, 7, 3
};
```



### 3. Create EBO

EBO is a buffer that stores indices used to draw vertices more efficiently
Uses an index array on how to reuse the vertices, rather than having to draw them twice

```cpp
unsigned int EBO;
glGenBuffers(1, &EBO);
```



### 4. Bind EBO to VAO

The order for this operation **does** matter
Tells OpenGL that this EBO belongs with this VAO
```cpp
glBindVertexArray(VAO);

glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, EBO);
glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);
```


### 5. Keep Vertex Attribute Setup

The attribute setup stays the same from last steps
```cpp
glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);
```



### 6. Draw Using Indices

Draw call is replaced in order to draw the correct triangles

```cpp
glDrawElements(GL_TRIANGLES, 36, GL_UNSIGNED_INT, 0);
```



### 7. Enable Depth Testing

This is **Important**
This determines which pixels are in front
Make sure to update clear to include the depth buffer bit

```cpp
glEnable(GL_DEPTH_TEST);
glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
```
