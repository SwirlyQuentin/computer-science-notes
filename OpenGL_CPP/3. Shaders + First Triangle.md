
### 1. Create Shader Files

Shaders written in [[GLSL]], a language that runs on the gpu

[[Vertex Shader]] (vertex.glsl):
```glsl
#version 410 core
layout (location = 0) in vec3 aPos;

void main() {
    gl_Position = vec4(aPos, 1.0);
}
```


[[Fragment Shader]] (fragment.glsl):
```glsl
#version 410 core
out vec4 FragColor;

void main() {
    FragColor = vec4(0.8, 0.3, 0.2, 1.0);
}
```


### 2. Minimal Shader Loader
Just a regular read file and output to a string

```cpp
#include <fstream>
#include <sstream>

std::string readFile(const char* path) {
    std::ifstream file(path);
    std::stringstream buffer;
    buffer << file.rdbuf();
    return buffer.str();
}
```


### 3. Compile Shaders

Converts the glsl into machince code the GPU can understand

```cpp
unsigned int vertexShader = glCreateShader(GL_VERTEX_SHADER);
std::string vertexCode = readFile("shaders/vertex.glsl");
const char* vShaderCode = vertexCode.c_str();
glShaderSource(vertexShader, 1, &vShaderCode, nullptr);
glCompileShader(vertexShader);
```

The same is done for the [[Fragment Shader]]


### 4. Link Shaders

- This step combines the vertex shader + fragment shader into one shader program

```cpp
unsigned int shaderProgram = glCreateProgram();
glAttachShader(shaderProgram, vertexShader);
glAttachShader(shaderProgram, fragmentShader);
glLinkProgram(shaderProgram);

glDeleteShader(vertexShader);
glDeleteShader(fragmentShader);
```


### 5. Compile Checking

Just checks the shaders compile and combine correctly

```cpp
int success;
char infoLog[512];

glGetShaderiv(vertexShader, GL_COMPILE_STATUS, &success);
if (!success) {
    glGetShaderInfoLog(vertexShader, 512, NULL, infoLog);
    std::cout << "Vertex Shader Error:\n" << infoLog << std::endl;
}
```


### 6. Create the Triangle Data

Data is in vertices 

```cpp
float vertices[] = {
     0.0f,  0.5f, 0.0f,
    -0.5f, -0.5f, 0.0f,
     0.5f, -0.5f, 0.0f
};
```


### 7. Setup [[VAO]] + [[Vertex Buffer Object (VBO)|VBO]]

Set up the buffers and add the triangle vertices into the buffer

```cpp
unsigned int VAO, VBO;

glGenVertexArrays(1, &VAO);
glGenBuffers(1, &VBO);

glBindVertexArray(VAO);

glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);
```

**This Line:**
```cpp
glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
```

 `0` - The attribute location
 - Attribute index 0 = position
 - Needs to match vertex shader

`3` - Number of components
- Each vertex has x, y, z

`GL_FLOAT` - Data type
- Vertices is an array of floats
- Data type is GL_FLOAT

`GL_FALSE` - Normalize?
- Telling not to remap values
- Used mostly for integers

`3 * sizeof(float)` - Stride
- Distance between vertices in memory
- Each vertex = 3 floats 
- 3 * the size of a float is the space between each vertex

`(void*)0` - Offset
- Where does this attribute start in the vertex
- starting at the beginning


### 8. Drawing The Triangle

Uses the combined shader program + VAO to draw triangles

```cpp
glClear(GL_COLOR_BUFFER_BIT);

glUseProgram(shaderProgram);
glBindVertexArray(VAO);
glDrawArrays(GL_TRIANGLES, 0, 3);
```
